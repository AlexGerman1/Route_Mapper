BUS ROUTE MAPPER

This project was completed as part of the "Introduction to Computer Science and Web Development Bootcamp" offered by Code Fellows. The project team would like to thank our instructor, Sam Hamm, and our TA's Dale Corns and Emre Surmeli.

Bus Route Mapper is implemented entirely in the browser with JavaScript. The map is created using the Leaflet.js library and the Mapbox and OpenStreetMap APIs. Bus route data is stored locally as a geoJSON file.

Leaflet is an open-source JavaScript library for mobile-friendly interactive maps. Tutorials and documentation can be found on their website.

We're using map tiles supplied by Mapbox for performance and beauty. As poor Code Fellows students, our map uses the free plan. If the map tiles don't appear to be coming from Mapbox it's because we ran out of service on our free tier.

All bus route data for this project came from King County GIS Center and is used under license. Data is provided "as-is" with no guarantee of accuracy or thoroughness. This bus route data was last updated by King County on 4/26/2015.
